## Overview

Dewa Smart Ticker is an enterprise-grade alert communication platform designed to send real-time alerts to client-installed systems within organizations. The system supports multiple alert types including Simple Message Alerts, Popup Alerts, Video Alerts, and Quiz Alerts, enabling organizations to communicate critical information effectively.

## Key Features

- **Multiple Alert Types**: Built a fully functional alert system supporting Simple Message Alerts, Popup Alerts, Video Alerts, and Quiz Alerts, each with unique display and interaction capabilities.
- **Real-time Communication**: Integrated APIs across the entire project to manage alert types, user data, and client interactions, ensuring seamless real-time alert delivery.
- **Multi-tenant Support**: Collaborated with the backend team to maintain smooth data flow and real-time alert delivery across multiple tenants, enabling organizations to manage alerts for different departments or locations.
- **Responsive Design**: Developed the complete frontend using Angular, TypeScript, HTML, and CSS, handling all UI designs and layouts with a focus on responsiveness and user experience.

## Technologies Used

- **Angular**: For building the main application framework and component architecture.
- **TypeScript**: For type-safe development and improved code maintainability.
- **HTML/CSS**: For creating responsive and visually appealing UI layouts.
- **RESTful APIs**: For managing alert types, user data, and client interactions.

## Challenges and Solutions

One of the main challenges was ensuring real-time alert delivery across multiple tenants while maintaining performance. This was solved by implementing efficient API integrations and optimizing the front-end performance for smooth user experience. Additionally, creating a unified system that supports multiple alert types required careful architectural planning to ensure scalability and maintainability.

## Outcome

The platform successfully enables organizations to send real-time alerts to client-installed systems, improving communication efficiency and ensuring critical information reaches the right audience promptly. The system has been tested and debugged to ensure flawless performance across multiple devices and screen sizes.
