## Overview

Orbantis Technologies Company Website is a modern, professional website built to showcase the company's comprehensive digital solutions. This project was developed using **Next.js** and **Tailwind CSS**, providing a fast, responsive, and visually appealing platform. The website highlights services including React & Django development, Shopify stores, WordPress sites, Laravel applications, mobile apps, AI chatbots, machine learning models, and more.

## Technologies Used

- **Next.js**: Primary framework for server-side rendering, routing, and optimal performance
- **Tailwind CSS**: Complete styling solution using utility-first CSS framework
- **React**: For building interactive components and dynamic content
- **Responsive Design**: Mobile-first approach using Tailwind CSS breakpoints

## Technical Implementation

The entire website is built using **Next.js** as the core framework, which provides:
- Server-side rendering (SSR) for improved performance and SEO
- Automatic code splitting for optimal bundle sizes
- Built-in routing and navigation system
- Image optimization and other performance enhancements

**Tailwind CSS** is used for all styling throughout the project:
- Utility-first approach for rapid UI development
- Responsive design with mobile-first breakpoints
- Consistent design tokens and spacing system
- Easy customization and theme configuration

## Key Features

- **Service Portfolio**: Comprehensive display of services including Web Development, Mobile Development, AI & Automation, and UI/UX Design, all styled with Tailwind CSS
- **Technology Stack Showcase**: Detailed presentation of technologies used (React, Next.js, Django, React Native, Flutter, AI/ML, Python, SQL, ETL)
- **Project Showcase**: Featured projects section highlighting e-commerce platforms, AI chatbots, and mobile banking applications
- **Modern Design**: Clean, professional design built entirely with Tailwind CSS utility classes, featuring smooth animations and interactive elements
- **Contact Integration**: Multiple contact options including WhatsApp, email, and phone for easy client communication
- **SEO Optimized**: Leveraging Next.js server-side rendering for optimal search engine visibility and performance

## Development Approach

The project follows modern web development best practices:
- **Next.js** for the React framework and server-side capabilities
- **Tailwind CSS** for all styling needs, creating a maintainable and scalable design system
- Component-based architecture for code reusability
- Performance optimization through Next.js built-in features
