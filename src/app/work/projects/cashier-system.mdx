## Overview

Cashier System is a companion application built to redeem loyalty rewards in real-time at the point of sale. The system works in conjunction with the main Loyalty Program System, allowing cashiers to validate and apply customer rewards during checkout, ensuring accurate and immediate point updates.

## Key Features

- **Real-time Redemption**: Developed functionality for redeeming earned points via coupon codes or direct balance checks, enabling instant reward application at the point of sale.
- **Synchronization**: Ensured smooth synchronization with the main loyalty backend for accurate real-time point updates, maintaining data consistency across systems.
- **API Integration**: Integrated APIs to validate, apply, and update redeemed rewards tied to the loyalty system, ensuring reliable communication between frontend and backend.
- **User Interface**: Created the UI and implemented business logic using Angular, TypeScript, Bootstrap, and CSS, providing an intuitive interface for cashiers to process redemptions efficiently.

## Technologies Used

- **Angular**: For building the application framework and component structure.
- **TypeScript**: For type-safe development and improved code quality.
- **Bootstrap**: For responsive UI components and layout system.
- **CSS**: For custom styling and visual design.
- **RESTful APIs**: For validating, applying, and updating redeemed rewards.

## Challenges and Solutions

The primary challenge was ensuring real-time synchronization between the cashier system and the main loyalty backend, especially during high-traffic periods. This was solved by implementing efficient API calls and proper error handling mechanisms. Additionally, creating a user-friendly interface that allows cashiers to quickly process redemptions required careful UX design and optimization of the workflow.

## Outcome

The Cashier System successfully enables real-time reward redemption at the point of sale, providing a seamless experience for both cashiers and customers. The system maintains accurate synchronization with the loyalty backend, ensuring reliable reward processing and point updates.
