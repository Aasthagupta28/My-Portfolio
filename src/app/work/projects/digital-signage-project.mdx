## Overview

Digital Signage Project is a centralized screen content management system that enables organizations to control and schedule media content on display screens. The system is used for various purposes including advertisements, government information displays, and corporate communications.

## Key Features

- **Content Scheduling**: Enabled content scheduling features for videos, images, and other media with options to set duration, orientation (landscape/portrait), and display priority.
- **Screen Management**: Integrated APIs to manage screen data, playlist configurations, and screen assignments, allowing administrators to control multiple displays from a central location.
- **Dynamic Playlists**: Built modular display templates and customizable playlists to create dynamic screen loops, ensuring engaging and varied content display.
- **Complete Frontend Development**: Developed the complete frontend using Angular, TypeScript, Bootstrap, and CSS, including all screen management interfaces and user controls.

## Technologies Used

- **Angular**: For building the main application framework and component-based architecture.
- **TypeScript**: For type-safe development and improved code quality.
- **Bootstrap**: For responsive UI components and layout system.
- **CSS**: For custom styling and visual enhancements.
- **RESTful APIs**: For managing screen data, playlist configurations, and screen assignments.

## Challenges and Solutions

One of the key challenges was creating a flexible scheduling system that supports various media types and orientations while maintaining performance. This was solved by implementing a modular architecture with reusable components. Additionally, managing multiple screens and playlists required efficient API design and state management to ensure smooth operation.

## Outcome

The Digital Signage system successfully enables organizations to manage and schedule content across multiple display screens, providing a centralized solution for digital communication. The system has been optimized for performance and tested to ensure reliable operation across different screen sizes and orientations.
