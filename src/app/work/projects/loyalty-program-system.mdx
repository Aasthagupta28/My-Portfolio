## Overview

Loyalty Program System is a food-tech platform that enables restaurants to register and manage customer loyalty programs with reward-based campaigns. The system allows restaurants to create engaging loyalty programs that reward customers based on order value or frequency, encouraging repeat business and customer retention.

## Key Features

- **Loyalty Program Management**: Implemented the full loyalty program functionality, assigning rewards based on order value or frequency, providing flexibility for different restaurant needs.
- **Reward System**: Defined and applied rules for point earning, coupon generation, and reward distribution, creating a comprehensive incentive system for customers.
- **Campaign System**: Developed a campaign system to send coupon codes to customers, which can be redeemed during future orders, enhancing customer engagement.
- **API Integration**: Integrated APIs to manage user registrations, order tracking, point calculation, and reward redemption, ensuring seamless data flow between frontend and backend.
- **Responsive Design**: Designed and developed the frontend using Angular, TypeScript, Bootstrap, and CSS, including responsive layouts and UI components that work across all devices.

## Technologies Used

- **Angular**: For building the main application framework and component architecture.
- **TypeScript**: For type-safe development and improved maintainability.
- **Bootstrap**: For responsive layouts and consistent UI components.
- **CSS**: For custom styling and visual design.
- **RESTful APIs**: For managing user registrations, order tracking, point calculation, and reward redemption.

## Challenges and Solutions

The main challenge was creating a flexible reward system that supports different business models while maintaining accurate point calculations and coupon management. This was addressed by implementing a rule-based system that allows restaurants to configure their own reward parameters. Additionally, ensuring real-time synchronization between the loyalty system and order processing required careful API design and state management.

## Outcome

The Loyalty Program System successfully enables restaurants to create and manage engaging customer loyalty programs, leading to increased customer retention and repeat business. The system has been tested and optimized to ensure reliable operation and accurate reward calculations.
