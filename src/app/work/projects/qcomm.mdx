## Overview

QComm (Quick Communication Tool) is a comprehensive communication platform that merges functionalities from DEWA Smart Ticker and Digital Signage. The system allows administrators to send alerts as well as control and configure client display screens, providing a unified solution for organizational communication and digital signage management.

## Key Features

- **Unified Platform**: Merged functionalities from DEWA Smart Ticker and Digital Signage into QComm, creating a single platform for both alert management and screen control.
- **Modern Technology Stack**: Rebuilding the user interface and core functionalities using React, Next.js, and CSS/Bootstrap, focusing on improved performance and scalability.
- **RESTful API Integration**: Implementing RESTful API integrations for alerts, screen control, and user management, ensuring efficient data flow and real-time updates.
- **UI/UX Enhancement**: Focused on improving UI/UX consistency, performance optimization, and component reusability across the system.
- **Scalable Architecture**: Working on modular, scalable frontend architecture for long-term maintainability, ensuring the system can grow with organizational needs.

## Technologies Used

- **React**: For building interactive user interfaces with component-based architecture.
- **Next.js**: For server-side rendering, improved performance, and better SEO.
- **TypeScript**: For type-safe development and improved code maintainability.
- **CSS/Bootstrap**: For responsive design and consistent styling.
- **RESTful APIs**: For alerts, screen control, and user management.

## Current Development Status

The project is currently in active development, with ongoing work to rebuild the user interface and core functionalities using modern technologies. The focus is on creating a more performant, scalable, and maintainable solution that provides a better user experience for administrators and end users.

## Challenges and Solutions

The main challenge is migrating from Angular to React/Next.js while maintaining all existing functionalities and ensuring a smooth transition. This is being addressed by carefully planning the migration, creating reusable components, and implementing a modular architecture that supports both current and future requirements.

## Expected Outcome

Once completed, QComm will provide a unified, modern platform for organizational communication and digital signage management, with improved performance, better user experience, and a scalable architecture that supports long-term growth and maintenance.
